# ADR 003: Supabase jako opcjonalna baza ciekawostek AI

## Status
Zaakceptowany

## Kontekst
Ciekawostki generowane przez AI były dotychczas przechowywane tylko w stanie strony. Użytkownik mógł je rozwinąć, ale nie miały trwałych linków.

## Decyzja
Dodajemy opcjonalną integrację z Supabase przez REST API. Po wygenerowaniu ciekawostki endpoint `/api/fact` próbuje zapisać ją do tabeli `ai_facts`. Jeśli zapis się uda, aplikacja może linkować do `/fact/ai-*`.

## Uzasadnienie
- Supabase REST API nie wymaga dodatkowego pakietu npm.
- Integracja działa po stronie serwera i nie ujawnia klucza klientowi.
- Aplikacja zachowuje fallback, jeśli baza nie jest skonfigurowana.

## Konsekwencje
- W Vercel trzeba ustawić zmienne `SUPABASE_URL` i `SUPABASE_SERVICE_ROLE_KEY`.
- Trzeba utworzyć tabelę `ai_facts`.
- Dopiero po konfiguracji bazy ciekawostki AI będą miały trwałe linki.
