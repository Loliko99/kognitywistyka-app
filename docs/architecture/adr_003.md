# ADR 003: Supabase jako opcjonalna baza ciekawostek AI

## Status
Zaakceptowany

## Kontekst
Ciekawostki generowane przez AI mogą mieć stałe linki tylko wtedy, gdy zostaną zapisane poza stanem klienta.

## Decyzja
Dodajemy opcjonalną integrację z Supabase przez REST API. Po wygenerowaniu ciekawostki endpoint `/api/fact` próbuje zapisać ją do tabeli `ai_facts`.

## Uzasadnienie
- Supabase REST API nie wymaga dodatkowego pakietu npm.
- Integracja działa po stronie serwera.
- Aplikacja zachowuje fallback, jeśli baza nie jest skonfigurowana.

## Konsekwencje
- Vercel musi mieć `SUPABASE_URL` i `SUPABASE_SERVICE_ROLE_KEY`, aby trwałe linki działały.
- Schemat tabeli jest zapisany w `docs/tech/supabase_ai_facts.sql`.
- Bez Supabase AI nadal działa, ale bez stałych linków.
