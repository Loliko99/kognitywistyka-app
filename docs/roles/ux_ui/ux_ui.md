# UX/UI

## Makiety
Makieta tekstowa strony głównej:

```text
Sticky nav
Hero: nowoczesne laboratorium poznania
Generator ciekawostek AI
Archiwum ciekawostek + filtry
Statystyki / sygnały tematyczne
Czym jest kognitywistyka?
Ścieżki kariery
Projekty, badania i technologie
CTA rekrutacyjne
```

## Przepływ Użytkownika
1. Użytkownik otwiera stronę główną.
2. Widzi hero i natychmiast poniżej sekcję ciekawostek.
3. Może wygenerować ciekawostkę AI albo przejrzeć archiwum.
4. Może rozwinąć ciekawostkę lub przejść na podstronę szczegółów.
5. Niżej poznaje kierunek, obszary nauki, ścieżki kariery i badania.

## Zasady UX
- Główna funkcja nie może być schowana na dole strony.
- Layout ma być mobile-first i czytelny.
- Styl: ciemniejsze laboratorium poznania, minimalizm, subtelny futurystyczny detal.
- Teksty mają być krótkie i łatwe do skanowania.
- Hover states i mikrointerakcje mają wspierać orientację, nie dominować interfejsu.
